Executive Summary

This report presents a comprehensive analysis of HCL Technologies (HCLTECH) stock data using Apache PySpark's RDD (Resilient Distributed Dataset) API. The dataset spans from August 2002 to April 2026 and contains daily trading records including Open, High, Low, Close prices, Adjusted Close, and Volume.
 
Five key insights were derived using distributed data processing operations such as map(), reduceByKey(), join(), sortBy(), and filter(). The analysis covers price trends by day and year, trading volume patterns, volatility, and market sentiment.

Conclusion

This PySpark RDD-based analysis of HCLTECH stock data reveals a compelling story of one of India's premier IT companies. The data confirms:
 
• Long-term wealth creation: 50x+ price appreciation over 24 years.
• Predictable seasonal patterns: October spikes in volume; Thursday strongest close.
• Event-driven volatility: Global macro events (COVID, 2008 crisis) drive extreme intraday swings.
• Positive market bias: More bullish sessions than bearish, supporting a buy-and-hold strategy.
 
PySpark's RDD API proved highly effective for this time-series financial analysis, enabling distributed computation patterns with concise Python lambda functions. The local mode on macOS M2 handled the full 24-year dataset efficiently.
